## Deployment (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select `main` branch, `/ (root)` folder
4. Click **Save**
5. Your site will be live at `https://yourusername.github.io/repo-name`
